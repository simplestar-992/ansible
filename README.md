## Ansible

A useful command-line tool for developers.

### Installation

```bash
git clone https://github.com/simplestar-992/ansible.git
cd ansible
go build -o ansible .
```

### Usage

```bash
./ansible --help
```

---

## Quick Start

```bash
./ansible [command]
```
