# ANSIBLE

**Useful CLI tool for developers**

## Features

- 🚀 Fast and efficient
- 📦 Single binary, no dependencies  
- 🔧 Easy to use
- 🔒 Secure by design

## Installation

```bash
git clone https://github.com/simplestar-992/ansible.git
cd ansible
go build -o ansible .
```

## Usage

```bash
./ansible --help
```
